FAQ

Is this a subscription?

No. $15 once. The license is valid forever and covers all Gudrun 1.x updates.

Where does the license file go?

Any one of these works:

  • %APPDATA%\Gudrun\gudrun.key (Windows) or ~/.config/gudrun/gudrun.key (Linux)
  • Next to the Gudrun executable
  • The Import button in the reminder dialog

Does Gudrun connect to the internet?

No. The license is verified locally with a cryptographic signature. No activation servers, no telemetry, no update pings. If you never visit this site again, nothing changes.

What happens when the trial ends?

After 30 days you get a reminder dialog at startup. Every feature keeps working — nothing locks, nothing is watermarked. The license removes the reminder. We think honest software sells itself.

How many machines can I use it on?

A personal license covers you, the person — install it on your own machines. For business use, email support@midnightsoftworks.com.

Can I make my own theme?

Yes — themes are single declarative files, applied instantly, no restart. Full guide with a complete template.

Windows says "unknown publisher" when I run the installer

That's Microsoft SmartScreen being cautious about new software that hasn't accumulated download reputation yet. Click "More info" → "Run anyway". Code signing to remove this warning is on the roadmap.
